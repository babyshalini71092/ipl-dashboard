# ipl-dashboard
IPL cricket data dashboard built with Python, pandas and Streamlit
<br>
This project uses the IPL Matches Dataset (2008–2025) from Kaggle.
1. Go to: https://www.kaggle.com/datasets/chaitu20/ipl-dataset2008-2025
2. Click the `IPL.csv` file → Download
3. Place it inside the `data/` folder as `IPL.csv`
Note: The dataset is not included in this repo due to GitHub's 
100MB file size limit.
